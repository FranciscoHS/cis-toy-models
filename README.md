This is the repo for my project for the Pivotal Vibeathon, investigating toy models of neural networks exhibiting computation in superposition (using fewer neurons than features by encoding multiple features per neuron and computing on them simultaneously).

The goal was to replicate the findings of [Compressed Computation is (probably) not Computation in Superposition](https://www.lesswrong.com/posts/ZxFchCFJFcgysYsT9/compressed-computation-is-probably-not-computation-in) (Bhagat et al.) (success!) and see if I can change their setup to actually elicit computation in superposition (there's evidence of success, but I'd want to dig into what the network's doing in a bit more depth before claiming it). All code written by Claude Opus 4.6.

If you want to read further, there's two .md files in this repository. writeup.md introduces the problem and presents the results obtained. It was generated by Claude based on my notes and I've only skimmed it, so take it with a grain of salt. note_dump.md has my raw notes taken in Obsidian as I worked on this. Low information density, I recommend having a look if and only if you're interested in how I think out loud and/or want to see all the brilliant insights Claude had access to.
